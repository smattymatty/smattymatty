## Founder of [Storm Developments](https://stormdevelopments.ca)

Canadian cloud infrastructure, built on open source and operated by us.

**[Storm Buckets](https://stormdevelopments.ca)** - S3-compatible object storage hosted in Canada, built on Garage. Point rclone, the AWS CLI, or Terraform at a bucket and the data stays on Canadian soil.

**[Storm Pulse](https://git.stormdevelopments.ca/official-public/storm-pulse)** - the open source, security-first agent that runs the servers Buckets sits on. Outbound-only, mTLS, HMAC-signed commands, zero listening ports. AGPLv3.
